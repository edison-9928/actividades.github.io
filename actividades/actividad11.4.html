CSS Selectors

Hay muchos tipos diferentes de selectors CSS que le permiten orientar las reglas a elementos especificos en un documento HTML.

Los selectores CSS distinguen entre mayusculas y minusculas, por lo que deben coincidir exactamente con los nombres de el elemento y los valores del atributo.

Tipos de selectores
Selector Significado Ejemplo
Universal Selector se aplica a todos los elementos del documento * {} se dirige a todos los elementos de la pagina
Type Selector Coincide con los nombres de el elemento h1, h2, h3 {} corresponde a los elementos <h1>, <h2> y <h3>
Class Selector Coincide con un elemento cuyo atributo de clase tiene un valor que corresponda con el espesificado despues del simbolo de punto "." . nota {} se dirige a cualquier elemento cuyo atributo de clase tenga un valor de nota
Id Selector Coincide con un elemento cuyo ID atributo tiene un valor que corresponda con el espefificado despues de el hash "#" #introduction {} se dirige al elemento cuyo atributo ID tiene un valor de introduction
